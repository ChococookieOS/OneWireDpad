# OneWireDpad

## Download
The newest arduino compatible library can be found under [releases](https://github.com/ChococookieOS/OneWireDpad/releases)
## Schematic
Dpad cable color | Pin functionality
-----------------|-------------------
Red              | VCC (3.3V - 12V)
Green            | Analog
White            | GND
